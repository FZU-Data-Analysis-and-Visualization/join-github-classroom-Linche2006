## 2. 代码块与公式
### （1）Python代码块
代码功能：读取销售数据并计算平均销售额
```python
import pandas as pd

# 读取数据
data = pd.read_csv("sales_data.csv")

# 计算销售额均值
avg_sales = data["销售额"].mean()
print(f"平均销售额：{avg_sales:.2f}万元")
